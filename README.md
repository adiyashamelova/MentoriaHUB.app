MentoriaHUB
Команда проекта
1. Шамелова Адия — Lead Development, Архитектура, AI Integration & Demo.
2. Орынбасар Айша — Pitching, Сценарий, Презентация & UX/UI Concept.

MentoriaHUB — это единая EdTech-экосистема для учеников старших классов (8–11/12 классы), которая структурирует образовательные возможности и заменяет хаос Telegram-каналов умной платформой для комплексной подготовки к поступлению в вузы. Проект разработан в рамках хакатона.

Живая демо-версия проекта на Vercel:
https://mentoriahubkz.vercel.app/

---

Проблематика, которую решает проект
1. Информационный хаос: Олимпиады, гранты и стажировки разбросаны по сотням чатов в мессенджерах.
2. Сложный выбор: Старшеклассникам тяжело найти программы, подходящие именно под их класс и личные интересы.
3. Жесткий график: Отсутствие гибких (self-paced) асинхронных курсов для совмещения с основной учебой и экзаменами.
4. Ручной менеджмент: Менторы тратят часы на ручные рассылки в мессенджерах и хаотичное обновление баз данных.
5. Упущенные дедлайны: Отсутствие системного трекинга личного прогресса и критических сроков подачи заявок.

Основной функционал (Модули платформы)

 1. Главная страница и Умный каталог
- Адаптивный дизайн: Полная поддержка трех языков (KZ/RU/EN) и ночной темы для комфортной работы в темное время суток.
- Smart Onboarding: Интерактивное анкетирование при первом входе (класс, интересы, цели) для автоматического тегирования профиля.
- Умный каталог: Автоматическое ранжурование и фильтрация конкурсов по классу, формату (онлайн/офлайн) и направлению. Поддержка быстрого поиска, добавления в «Избранное» и подачи заявки в один клик.
- Интегрированный FAQ: Блок быстрых ответов на частые вопросы новых пользователей.

 2. Асинхронный LMS-модуль (Курсы)
- Полностью автономное обучение в собственном темпе (SAT, математика, языки) с наглядным прогресс-баром курса.
- Структурированная страница лекции: Встроенный кастомный видеоплеер и три удобные вкладки с материалами (краткий конспект, формулы/файлы и мини-задания).
- Интерактивные тесты: Модуль моментальной проверки знаний с автоматическим подсчетом баллов.
- Геймификация: Эффектная анимация успешного завершения курса и автоматическая генерация именного сертификата после финального теста.

 3. Личный кабинет с AI-Ментором
- Персональный дашборд с профилем ученика, его баллами и сохраненными программами.
- Центр контроля дедлайнов: Тайм-трекер для отслеживания критических сроков и игровой лидерборд (рейтинг студентов).
- Индивидуальный Roadmap: Интерактивная дорожная карта развития школьника под конкретный класс.
- Интеграция с AI Gemini: Встроенный чат-ассистент на базе Google Gemini API для персональных образовательных консультаций 24/7.

 4. Панель администратора (Админка)
- No-code CRUD-интерфейс: Добавление, редактирование и удаление курсов/конкурсов одновременно на трех языках без участия разработчиков.
- Гибкие настройки карточек: Управление целевыми классами (8–12), категориями (STEM, Coding, Business и др.), форматами и точными датами дедлайнов.
- Аналитический дашборд: Мониторинг активности платформы в реальном времени (количество юзеров, пройденных тестов, выданных сертификатов).

---

 Стек технологий и архитектура
- Frontend: HTML5, CSS3 (с использованием CSS-переменных для бесшовной смены тем), JavaScript (ES6+).
- Data Management: Единый структурированный реестр данных (база данных на основе JSON / LocalStorage).
- AI Integration: Google Gemini API (интеллектуальный чат-ассистент).
- UI/UX: Адаптивная верстка под мобильные и десктопные устройства, кастомный медиаплеер и табы.

 Установка и запуск (Локально)

1. Склонируйте репозиторий:
```bash
git clone [https://github.com/adiyashamelova/MentoriaHUB.git](https://github.com/adiyashamelova/MentoriaHUB.app.git) mentoria-hub
```
 2. Перейдите в директорию проекта:
```bash
cd mentoria-hub

```
 3. Откройте файл index.html в вашем браузере (или запустите через расширение Live Server в VS Code).
